# UI Reference Notes

The screenshot in `design/ui-reference.png` shows a dense dark table dashboard. Important visual elements to copy:

- Very dark background, almost black.
- Header area with logo/title on left and stat pills on right.
- Thin horizontal separator lines.
- Top nav tabs with active underline.
- Filter row with compact input boxes and selects.
- Table with small row height and muted column headers.
- Category badges in purple/green/blue.
- Subcategory/status badges as slate rounded rectangles.
- Positive numbers in green.
- Pagination on the right side of filter row.
- Minimal borders; row separators are subtle.

Do not copy the exact GoodAIList branding. Use the style only.
